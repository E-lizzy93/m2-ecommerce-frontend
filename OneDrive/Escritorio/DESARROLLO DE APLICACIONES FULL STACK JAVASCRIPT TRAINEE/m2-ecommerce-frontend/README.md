# M2 — E-commerce Frontend

## Descripción

 El siguiente Proyecto  es parte del módulo 2 creando una tienda online.
El objetivo de  es presentar una interfaz navegable que simule el flujo básico de una tienda online,
incluyendo iniciel home, listado de productos y detalle de producto.

El diseño se basa en referencias visuales entregadas como ejemplo durante el desarrollo de las actividades del modulo , priorizando consistencia visual y jerarquía de información..

## Tecnologías
- HTML5
- CSS3
- Bootstrap por CDN

## Ejecutar el proyecto
- Opción 1: abrir `index.html` en el navegador.
- Opción 2 (recomendada): usar una extensión de servidor local (Live Server en VS Code).

## Funcionalidades

 - Navbar para navegar por la página
- Home con productos algunos destacados.
- Listado en formato grid con filtros visuales (orden por precio y nombre).
- Tarjetas de productos con botón “Agregar”.
- Página de detalle de producto  principal, con imagen de referencia, precio, disponibilidad, descripción y CTA.
- Footer con información de la página.
